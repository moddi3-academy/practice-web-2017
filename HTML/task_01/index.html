Заняття № 1

Тема. Вступ в HTML. Структура HTML-коду. Теги

Мета: ознайомитись з основними відомостями про HTML (HyperText Markup Language, мова розмітки гіпертексту), іструментарієм; вивчити основні теги, синтаксис HTML, структуру веб-сторінки.

Практичне завдання

Створити найпростішу веб-сторінку, яка міститиме інформацію по першому практичному заняттю (див. зразок нижче).
Створити власну папку на диску d: в папці students.
Використовувати програму notepad++ (d:\programs\notepad++).
В <title> прописати текст «Заняття №1» (без кавичок).
Для задання кольору фону сторінки використати атрибут bgcolor.
Для створення заголовків використовуйте теги <h1>, <h2> та <h3>.
Для створення абзаців використовуйте тег <p>.
Для переходу на наступний рядок (розрив рядка) без створення абзацу використовуйте одиночний тег.
Щоб зробити текст жирним використовуйте тег <b> .
